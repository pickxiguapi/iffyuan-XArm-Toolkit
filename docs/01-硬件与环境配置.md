# 硬件与环境配置

> 这篇解决的问题：新人拿到机械臂，怎么接线、配网、装环境、第一次上电。

## 硬件清单与接线

### 机械臂 (XArm 7)

1. **电源**：接 220V 电源，开机开关在底座后方
2. **网线**：机械臂控制器 → 路由器（或直连电脑），确保在同一网段
3. **默认 IP**：`192.168.31.232`（出厂或通过 xArm Studio 修改）
4. **验证连接**：
   ```bash
   ping 192.168.31.232
   ```

> **注意**：机械臂上电后会有一个自检过程（约 5 秒），自检完成前不要发送任何指令。

### 相机

| 相机 | 型号 | Serial | 安装位置 | USB 接口 |
|------|------|--------|---------|---------|
| 臂上相机 | Intel D435i | `327122075644` | 机械臂末端法兰 | USB 3.0 |
| 固定相机 | Intel L515 | `f1271506` | 工作台固定支架 | USB 3.0 |

- **必须用 USB 3.0 口**，USB 2.0 带宽不够会丢帧
- 两个相机插同一台电脑
- 验证相机是否被识别：
  ```bash
  # 方法一：用 realsense-viewer (如果装了 librealsense)
  realsense-viewer

  # 方法二：用我们的代码
  python -c "import open3d as o3d; print(o3d.t.io.RealSenseSensor.list_devices())"
  ```

### SpaceMouse (3DConnexion)

- USB 连接电脑即可
- Linux 下需要安装 `spacenavd` 驱动：
  ```bash
  sudo apt install spacenavd
  sudo systemctl start spacenavd
  ```
- 验证：
  ```bash
  # 检查设备是否识别
  ls /dev/input/event* | head -5
  ```

## 软件环境安装

### Python 环境

推荐 Python 3.9+，用 conda 管理：

```bash
conda create -n xarm python=3.10
conda activate xarm
```

### 安装依赖

```bash
cd iffyuan-XArm-Toolkit

# 安装本包（开发模式）
pip install -e .

# 或者手动装核心依赖
pip install numpy opencv-python open3d zarr h5py pytransform3d xarm-python-sdk termcolor
```

### 依赖说明

| 包名 | 用途 | 必须？ |
|------|------|--------|
| `xarm-python-sdk` | 机械臂 SDK（官方） | 是 |
| `numpy` | 数据计算 | 是 |
| `opencv-python` | 图像处理、显示 | 是 |
| `open3d` | RealSense 相机驱动 + 点云 | 用相机时需要 |
| `pytransform3d` | 旋转/坐标变换 | 是 |
| `zarr` | 数据采集存储 | 采集数据时需要 |
| `h5py` | 数据读取 | 可选 |
| `termcolor` | 终端彩色输出 | 可选 |

## 网络配置

### 机械臂 IP 设置

机械臂默认 IP 为 `192.168.31.232`。如果需要修改：

1. 用 xArm Studio 连接（下载地址：[UFACTORY 官网](https://www.ufactory.cc/download-xarm-robot)）
2. 在 Settings → Network 中修改 IP
3. 修改后同步更新代码中的 `addr` 参数

### 电脑网络要求

- 电脑 IP 需要和机械臂在同一网段（如 `192.168.31.xxx`）
- 如果电脑通过 WiFi 上网，机械臂走有线，需要确保路由器能互通
- 建议：电脑用一个网口直连机械臂，WiFi 上网，互不干扰

## 首次上电流程

1. **上电机械臂**：打开底座电源开关，等待自检完成（约 5 秒）
2. **ping 测试**：`ping 192.168.31.232` 确认连通
3. **插上相机**：两个 USB 3.0
4. **运行 mock 测试**：先不连真机，确认代码没问题
   ```bash
   python scripts/web_control_demo.py --mock
   ```
5. **运行真机测试**：
   ```bash
   python scripts/web_control_demo.py
   ```
   打开 `http://localhost:8080`，按 R 复位，然后用 WASD 小幅度试探

> **第一次操作务必注意**：
> - 确保机械臂周围无障碍物
> - 手放在急停按钮上（机械臂底座上的红色按钮）
> - 先用 Web Demo 的小步长（默认 2mm/步）熟悉方向
