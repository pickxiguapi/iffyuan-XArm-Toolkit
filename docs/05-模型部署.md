# 模型部署

> **状态**：deploy 模块正在开发中，本文档描述整体架构和接口约定，待实现后补充完整代码示例。

## 部署架构

```
VLA 模型推理 (GPU)
     ↓ action (6D 关节角 or 末端位姿)
XArmEnv (absolute_joint 或 absolute_eef)
     ↓
机械臂执行
     ↓
双相机 + 力传感器 → observation → 下一帧推理输入
```

## 部署时的 XArmEnv 配置

### Pi0 / Pi0.5（关节角输出）

```python
env = XArmEnv(
    action_mode="absolute_joint",  # Pi0 输出关节角
    use_force=False,                # 根据模型需求决定
    initial_gripper_position=840,
)
```

### 笛卡尔位姿输出的模型

```python
env = XArmEnv(
    action_mode="absolute_eef",    # 模型输出目标位姿
    use_force=False,
    initial_gripper_position=840,
)
```

## 推理循环（伪代码）

```python
import time

env = XArmEnv(action_mode="absolute_joint", use_force=False)
cam_arm = RealsenseEnv(serial="327122075644", mode="rgb")
cam_fix = RealsenseEnv(serial="f1271506", mode="rgb")

obs = env.reset()

# 推理主循环（~10 Hz）
while not done:
    # 1. 采集观测
    arm_rgb = cam_arm.step()["rgb"]
    fix_rgb = cam_fix.step()["rgb"]
    joint_angles = obs["servo_angle"]
    gripper_pos = obs["gripper_position"]

    # 2. 模型推理
    action = model.predict(arm_rgb, fix_rgb, joint_angles, gripper_pos)
    # action: [j1, j2, j3, j4, j5, j6] + gripper

    # 3. 执行
    obs = env.step(
        action=action[:6],
        gripper_action=action[6],  # 0 或 840
        joint_speed=0.5,
    )

    # 4. 频率控制
    time.sleep(0.1)  # ~10 Hz
```
